## 🔮 Version 2.0 — Intelligence Upgrade (Q3 2026)

### 🧠 Large Language Model Integration
- **OpenAI GPT-4 API** integration for production-grade responses
- **Google Gemini API** as an alternative AI backend
- **Local LLM support** via Ollama (Llama 3, Mistral) for offline AI
- Hybrid mode: rule-based fallback + LLM for unrecognized queries
- Context window management for long conversations
- Token usage tracking and cost monitoring

### 🎤 Full Voice Interaction
- **Real-time Speech-to-Text** using Google Cloud Speech API / Whisper
- **Text-to-Speech** output using Amazon Polly or Google TTS
- Wake word detection ("Hey NeuraBot")
- Voice command shortcuts (e.g., "Hey NeuraBot, open analytics")
- Support for Hindi speech recognition
