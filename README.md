## рҹ“Ң Project Overview

**NeuraBot AI** is a complete, professional-grade AI Conversational Assistant built as a **Final Year B.Tech Computer Science Project**. It demonstrates real-world implementation of:

- вң… **Natural Language Processing (NLP)** вҖ” Tokenization, intent detection, sentiment analysis
- вң… **Rule-Based AI Engine** вҖ” 16 intent categories, fuzzy knowledge matching
- вң… **Knowledge Base Management** вҖ” 30+ seeded FAQ articles, CRUD operations
- вң… **Premium JavaFX GUI** вҖ” 11 animated screens inspired by ChatGPT & Gemini
- вң… **MVC Architecture** вҖ” Clean separation of concerns with design patterns
- вң… **Data Persistence** вҖ” JSON-based storage (Gson) for all app data

> **"NeuraBot AI looks and feels like a commercial AI SaaS product вҖ” not a typical college project."**

## рҹҡҖ Key Features

### рҹӨ– AI & NLP Engine
- **16-class Intent Detection** вҖ” Greeting, Question, Farewell, SmallTalk, Help, Joke, Compliment, Complaint, etc.
- **3-level Sentiment Analysis** вҖ” Positive / Neutral / Negative with lexicon-based scoring
- **Keyword Extraction** вҖ” Stop-word filtering, fuzzy matching with confidence threshold (вүҘ 0.25)
- **5 AI Personalities** вҖ” Friendly, Professional, Teacher, Coder, Expert
- **30+ Knowledge Base Articles** вҖ” Pre-seeded across 10 categories

### рҹ’¬ Premium Chat Interface
- Animated message bubbles with **FadeTransition + TranslateTransition**
- **Typing indicator** with bouncing animated dots
- Conversation **session history** with sidebar navigation
- Smart **suggestion chips** for guided interaction
- Real-time **sentiment display** in status bar

### рҹ“ҡ Knowledge Base System
- **10 categories**: AI, Java, Algorithms, Machine Learning, Database, OS, Networks, DSA, Python, General
- Admin CRUD вҖ” Add / Edit / Delete FAQs
- Instant search with live filtering
- Hit count tracking for analytics

### рҹ“Ҡ Analytics Dashboard
- **Bar chart** вҖ” 7-day conversation volume (custom canvas rendering)
- **Pie chart** вҖ” Sentiment distribution (Positive / Neutral / Negative)
- **Top Topics** leaderboard with progress bars
- Export reports to **TXT** and **CSV**

### рҹ”’ Authentication System
- **SHA-256** password hashing
- Role-based access вҖ” **Admin** vs **User**
- Animated login with shake effect on error
- Session tracking with activity logs

### рҹ‘‘ Admin Panel
- User management вҖ” Promote / Delete accounts
- **Bot Training** вҖ” Add Q&A pairs to live knowledge base
- **Activity Log viewer** with real-time search

### вҡҷ Settings & Customization
- **Dark / Light theme** вҖ” instant switch with Observer pattern
- AI Personality selector
- Font size slider (12вҖ“20px)
- Animation speed control
- Auto-save, notifications, language toggle

| Requirement | Version | Check Command |
|-------------|---------|--------------|
| Java JDK | **21 or 25 (LTS recommended)** | `java -version` |
| Apache Maven | **3.8+** | `mvn -version` |

### Clone & Run

```bash
# 1. Clone the repository
git clone https://github.com/MohammadSakibAhmad0874/CodeAlpha_NeuraBot-AI-Intelligent-Conversational-Assistant.git
cd CodeAlpha_NeuraBot-AI-Intelligent-Conversational-Assistant

# 2. Build the project
mvn clean package -DskipTests

# 3. Run the application
mvn javafx:run
```

## рҹ§  NLP Pipeline

```
User Input Text
      в”Ӯ
      в–ј
 NLPProcessor.tokenize()         вҶ’  ["what", "is", "machine", "learning"]
      в”Ӯ
      в–ј
 NLPProcessor.extractKeywords()  вҶ’  ["machine", "learning"]  (stop-words removed)
      в”Ӯ
      в–ј
 IntentDetector.detect()         вҶ’  IntentType.DEFINITION_REQUEST
      в”Ӯ
      в–ј
 SentimentAnalyzer.analyze()     вҶ’  Sentiment.POSITIVE / NEUTRAL / NEGATIVE
      в”Ӯ
      в–ј
 KnowledgeBase.findBestMatch()   вҶ’  FAQ match (confidence вүҘ 0.25)
      в”Ӯ
      в–ј
 ChatbotEngine.generateResponse() вҶ’  Apply personality mode
      в”Ӯ
      в–ј
 Bot Response  вҶ’  Displayed with FadeIn animation
```

## рҹ“җ Design Patterns Used

| Pattern | Implementation |
|---------|---------------|
| **Singleton** | `DatabaseManager`, `KnowledgeBase` |
| **Observer** | `ThemeManager` notifies all registered scenes |
| **MVC** | View (JavaFX) вҶ’ AI Engine вҶ’ Database (JSON) |
| **Strategy** | AI personality modes (Friendly/Professional/Teacher/Coder/Expert) |
| **Factory** | `NotificationManager.showToast()` creates styled popups |

---

## рҹ“Ҳ OOP Principles Applied

- **Encapsulation** вҖ” Private fields + public getters/setters in all 6 model classes
- **Abstraction** вҖ” AI engine, NLP, and UI completely decoupled
- **Polymorphism** вҖ” `Message.Sender`, `ActivityLog.LogType`, `Message.Sentiment` enums
- **Inheritance** вҖ” JavaFX scene graph hierarchy, stage management
